# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in BookBed, please report it responsibly:

**Email:** security@bookbed.io (ili tvoj email)

### What to include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

### Response Timeline:
- **Acknowledgment:** Within 48 hours
- **Initial Assessment:** Within 7 days
- **Resolution:** Depends on severity (Critical: 24-48h, High: 7 days, Medium/Low: 30 days)

### What to expect:
- We will acknowledge your report promptly
- We will keep you informed of our progress
- We will credit you (if desired) when the fix is released

**Please do not disclose the vulnerability publicly until we have addressed it.**
