# Lottie Animations

This folder contains Lottie animation files for the booking flow.

## Required Files

### success.json
- **Usage:** Success screen celebration animation
- **Type:** Success/checkmark/celebration animation
- **Duration:** 2-3 seconds recommended
- **Loop:** No (plays once)

## Where to Get Lottie Files

### Free Sources:
1. **LottieFiles** (Recommended)
   - URL: https://lottiefiles.com
   - Search for: "success", "checkmark", "celebration"
   - Filter: Free, JSON format
   - License: Check individual animation licenses

2. **Lordicon**
   - URL: https://lordicon.com
   - Many free animated icons

3. **IconScout**
   - URL: https://iconscout.com/lottiefiles
   - Mix of free and premium

## How to Add Animations

1. Download the `.json` file from LottieFiles
2. Rename it to `success.json`
3. Place it in this folder (`assets/animations/`)
4. The app will automatically load it

## Animation Requirements

- **Format:** JSON (Lottie format)
- **Size:** Under 100KB recommended
- **Colors:** Should work with primary brand colors
- **Background:** Transparent preferred

## Recommended Success Animations

Search these IDs on LottieFiles:
- `lf20_success` - Simple checkmark
- `lf20_jrbw1pdp` - Confetti celebration
- `lf20_touohxv0` - Success with particles

## Testing

The success screen has a fallback icon if the Lottie file is not found, so the app will work even without the animation file. However, for the best user experience, please add the `success.json` file.

## License Notes
