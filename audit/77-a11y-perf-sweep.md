## §2 — a11y wins (IconButton tooltips)

12 IconButton sites gained screen-reader labels via `tooltip:` param. Material's `Tooltip` widget exposes a `Semantics(label:)` envelope, so screen readers (NVDA, JAWS, VoiceOver, TalkBack) announce the tooltip text on focus.

### 2.1 Widget guest form — `guest_count_picker.dart`

`_GuestRow` inc/dec IconButtons (adults / children / pets):

```dart
IconButton(
  onPressed: canDecrement ? onDecrement : null,
  tooltip: '$label −',  // ← was missing — now reads "Odrasli −" / "Adults −"
  icon: Icon(Icons.remove_circle_outline, ...),
),
```

Count `Text` wrapped in `Semantics(label: '$label: $count', excludeSemantics: true, ...)` — screen reader now announces "Adults: 2" instead of just "2".

### 2.2 Widget calendar headers

`year_calendar_widget.dart:235,264` — chevron buttons tooltip = target year.
`month_calendar_widget.dart:221,261` — chevron buttons tooltip = target month (`DateFormat.yMMM(locale).format(_previousMonth)`).

Users now know what date they'll land on before tapping. Especially useful on small screens where the chevron icons are 16×16 with 28×28 constraints (iPhone SE compact layout).

### 2.3 Widget additional services — `additional_services_widget.dart:225,263`

Quantity ± per service:

```dart
IconButton(
  icon: Icon(Icons.remove, ...),
  tooltip: '${service.name} −',  // e.g. "Kasni check-in −"
  ...
),
```
