PROMPT 14: Shared Widgets Library

Kreiraj reusable widgets za consistent UI:

1. BUTTONS (`lib/shared/widgets/buttons/`):

   a) `primary_button.dart`:
      - Full-width sa accent color
      - Loading state indicator
      - Disabled state
      - onPressed callback

   b) `secondary_button.dart`:
      - Outlined style
      - Hover effects

   c) `icon_button_widget.dart`:
      - Circular sa iconicom
      - Tooltip

2. INPUTS (`lib/shared/widgets/inputs/`):

   a) `custom_text_field.dart`:
      - Label, hint, prefix/suffix icons
      - Validation error display
      - Password visibility toggle
      - Focus state animations

   b) `date_picker_field.dart`:
      - Read-only text field
      - Calendar icon
      - Opens date picker on tap

   c) `dropdown_field.dart`:
      - Custom styling
      - Search functionality

   INPUT FIELDS (Original):
