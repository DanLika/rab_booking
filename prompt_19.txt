PROMPT 19: Deployment & DevOps

Pripremi projekat za production deployment:

1. ENVIRONMENT CONFIGURATION:

   a) `.env` files:
      - .env.development
      - .env.staging
      - .env.production

      Variables:
      - SUPABASE_URL
      - SUPABASE_ANON_KEY
      - STRIPE_PUBLISHABLE_KEY
      - API_BASE_URL

   b) `lib/core/config/env_config.dart`:
      Flutter-env-config loader

2. BUILD CONFIGURATIONS:

   a) Android (`android/app/build.gradle`):
      - Release signing config
      - ProGuard rules
      - Build variants (dev, prod)

   b) iOS (`ios/Runner/Info.plist`):
      - Bundle ID configurations
      - Schemes setup
      - App Transport Security

   c) Web (`web/index.html`):
      - Meta tags
      - PWA manifest
      - Service worker setup

3. CI/CD PIPELINES:

   a) `.github/workflows/build.yml`:
      ```yaml
      # Build workflow za sve platforme
      # Run tests
      # Code analysis
      # Build APK/IPA/Web
      # Deploy to stores/hosting
      ```

   b) Fastlane setup (iOS/Android):
      - Automated screenshots
      - Beta distributions
      - Store submissions

4. HOSTING:

   a) Web:
      - Firebase Hosting / Vercel
      - Custom domain setup
      - SSL certificates

   b) Backend:
      - Supabase project setup (production)
      - Database backups
      - Edge functions deployment

5. MONITORING & ANALYTICS:

   - Firebase Analytics integration
   - Crashlytics setup
   - Performance monitoring
   - User analytics events

6. DOCUMENTATION:

   - DEPLOYMENT.md sa step-by-step instructions
   - Architecture diagrams
   - API documentation
   - User guides

   Kreiraj complete deployment checklist sa:
   - Pre-deployment tasks
   - Build configurations
   - Testing requirements
   - Deployment steps (Android, iOS, Web)
   - Post-deployment verification
   - Rollback procedures

DELIVERABLES:
- Environment configuration files (.env.development, .env.staging, .env.production)
- EnvConfig loader (lib/core/config/env_config.dart)
- Android release signing config & ProGuard rules
- iOS bundle ID configurations & schemes
- Web PWA manifest & service worker
- GitHub Actions CI/CD workflow (build.yml)
- Fastlane configuration (iOS/Android)
- Firebase Hosting / Vercel deployment setup
- Supabase production project setup
- Firebase Analytics & Crashlytics integration
- DEPLOYMENT.md documentation
- Complete deployment checklist (docs/DEPLOYMENT_CHECKLIST.md)
- Architecture diagrams
- API documentation
